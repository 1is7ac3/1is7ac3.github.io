Cajero Automático. JavaScript

Este programa esta escrito en JavaScript posee el funcionamiento de un cajero automático, solo función de retiro se utilizan los botones para seleccionar la cantidad, borrar datos o canelar la operación. al procesar se pueden obtener tres resultados:

1. Retiro procesado, da como resultado la cantidad solicitada con una explicación detalla de los billetes suministrados.
2. Monto no puede ser dispensado. se obtiene este resultado si se introduce una cantidad que no puede ser dispensada en billetes.
3. si solicitamos una cantidad por encima de la que posee el cajero se imprime: "Cajero no disponible para retiros".
